# 形参和实参值
```java
// 定义一个加法方法
public static int add(int x, int y) { // x和y是形参：定义时声明需要两个int类型的参数
    return x + y; // 形参在方法内部使用，代表接收的实参值
}

public static void main(String[] args) {
    int a = 3;
    int b = 5;
    
    // 调用add方法时，传入的a和b（或5和8）是实参：具体的数值/变量
    int result1 = add(a, b); // 实参是变量a、b（值为3、5）
    int result2 = add(5, 8); // 实参是直接的数值5、8
    
    System.out.println(result1); // 输出8
    System.out.println(result2); // 输出13
}
```
