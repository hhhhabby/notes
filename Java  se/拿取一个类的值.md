# 修饰符
![[40d7646bff4b7d0ee53e16f39222232e.png]]

Public：
```java
        s.age = 18; // 直接访问public变量并赋值
        System.out.println(s.age); // 直接访问，输出18
```

Static：

```java
        Tool.version = 2; // 直接通过  类名     访问静态变量
        System.out.println(Tool.version); // 输出2
```

Protect:

```java
class Parent {
    protected String hobby; // protected变量
}

class Child extends Parent {
    public void printHobby() {
        hobby = "读书"; // 子类中直接访问父类的protected变量
        System.out.println(hobby); 
    }
}

public class Test {
    public static void main(String[] args) {
        Child c = new Child();
        c.printHobby(); // 输出"读书"
    }
}
```



Private:
```java
class Person {
    private String name; // private变量，外部无法直接访问

    // 提供public的getter方法，用于获取name的值
    public String getName() {
        return name; 
    }

    // 通常还会搭配setter方法用于赋值（setXxx()）
    public void setName(String n) {
        name = n; 
    }
}

public class Test {
    public static void main(String[] args) {
        Person p = new Person();
        p.setName("张三"); // 通过setter赋值
        String name = p.getName(); // 通过getter获取值
        System.out.println(name); // 输出"张三"
    }
}
```


反射：

```java
import java.lang.reflect.Field;

class Cat {
    private String color = "白色"; // private变量
}

public class Test {
    public static void main(String[] args) throws Exception {
        Cat cat = new Cat();
        // 通过反射获取private变量
        Field field = Cat.class.getDeclaredField("color");
        field.setAccessible(true); // 强制开启访问权限
        String color = (String) field.get(cat); // 获取值
        System.out.println(color); // 输出"白色"
    }
}
```
