### 3.4 justify-content属性
> - flex-start（默认值）：左对齐
> - flex-end：右对齐
> - center： 居中


######    靠左靠右
```css
  display: flex; /* 开启弹性布局 */
  justify-content: space-between; /* 左右两端对齐 */
  padding: 10px; /* 内部上下间距 */
```

```
> - space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。
