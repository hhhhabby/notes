# 图片数字识别

### 前言：
传统数字识别 OCR、大模型识别与 YOLOv8 的差异及 YOLOv8 优势

1. **传统数字识别 OCR 的 api 接口**
    - **局限性**：然而，当面对复杂背景、倾斜扭曲的数字、多种字体混合或存在遮挡情况时，传统 OCR 的性能会大幅下降。它缺乏对上下文语义的理解，难以处理非标准字体或手写数字的多样性。
2. **大模型识别**
    - **局限性**：但大模型往往参数众多，模型体积庞大，对计算资源和硬件要求极高，推理速度相对较慢，在实时性要求高或硬件资源有限的场景下不太适用。同时，训练大模型需要大量的数据和高昂的成本。
3. **YOLOv8 优势**


    - **免费**：YOLOv8 是基于 YOLO 系列的目标检测模型，在保持高精度的同时，具备卓越的实时检测能力。它采用了高效的网络架构设计，如改进的 Backbone、Neck 和 Head 结构，能够快速处理图像，适用于如自动驾驶中的交通标志数字识别、工业生产线上产品编号实时检测等场景，满足对实时性的严格要求。
    - **多目标检测**：可以同时检测图像或视频中的多个数字目标，无需像传统 OCR 那样针对单个字符逐一识别，大大提高了检测效率。在复杂场景中，能快速定位并识别不同位置、不同大小的数字，例如在大型商场的价格标签、停车场的车位编号等场景下，可快速准确地识别多个数字目标。
    - **速度快**：api 和调用火山大模型进行识别大约 3 s 左右。且无法处理

    - **灵活部署**：支持多种硬件平台，包括 CPU、GPU 以及一些边缘计算设备，方便在不同场景下进行部署，无论是在高性能服务器上进行大规模数据处理，还是在资源受限的嵌入式设备上实现实时检测，都能较好地适配。
