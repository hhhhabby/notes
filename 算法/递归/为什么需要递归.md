
### 1. 递归实现

#### 代码示例（Java）：

java

运行

```java
public int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}
```

- **时间复杂度**：O (n)
    
    递归过程中，每个整数 `0~n` 都会被计算一次，共执行 `n+1` 次操作，因此时间复杂度为 **O(n)**。
    
- **空间复杂度**：O (n)
    
    递归调用会在栈中保存 `n` 层函数调用的上下文（每个 `factorial(n)` 都会占用栈空间），因此空间复杂度为 **O(n)**。
